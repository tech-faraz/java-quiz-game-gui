# 🎮 Java Quiz Game (GUI)

## 📌 Description

This is an interactive **OOP-based GUI Quiz Game** developed in Java.
The game allows users to answer multiple-choice questions with a timer, calculate scores, and use a special **50-50 lifeline** to increase their chances of winning.

---

## 🎯 Features

* 🧠 Multiple-choice quiz questions
* ⏱️ Timer for each question
* 📊 Automatic score calculation
* 🎨 Graphical User Interface (GUI)
* 🔄 Next button for navigation
* 🎯 **50-50 Lifeline** (removes two incorrect options)
* 🎉 Result screen with final score

---

## 🧠 Special Feature

### 🎯 50-50 Lifeline

The player can use a 50-50 lifeline to eliminate two incorrect options, making it easier to choose the correct answer.
👉 This lifeline can be used **only once per game**.
